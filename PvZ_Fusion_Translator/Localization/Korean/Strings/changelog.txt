1. 새로운 식물들
- 일반 식물:
 - 얼음 랜턴 (얼음 버섯 + 플랜턴)
 - 바다 감자 지뢰 (감자 지뢰 + 바다 버섯)
 - 바다 감자 어뢰 (바다 버섯 + 감자 지뢰)
 - 파멸 랜턴 (파멸 버섯 + 플랜턴)
 - 양배추 대포 (양배추 펄트 ∞ [양배추 펄트] ∞ 양배추 펄트)
 - 스쿼시 멜론 (스쿼시 + 멜론 펄트)
 - 체리 스쿼시 (체리 폭탄 + 스쿼시)
 - 파멸 스쿼시 (파멸 버섯 + 스쿼시)
 - 버터 버섯 (가스 버섯 + 옥수수 펄트)
 - 연발 리피터 (콩슈터 + 삼발슈터)
 - 해바라기 블로버 (해바라기 + 블로버)
 - 호박 버섯 (뻐끔 버섯 + 호박)
 - 거품 호박 (가스 버섯 + 호박)
 - 수호박 (호박 + 멜론 펄트)
 - 스쿼시 지뢰/감자 스쿼시 (감자 지뢰 + 스쿼시)
 - 파멸 마늘 (파멸 버섯 + 마늘)
 - 큰별 버섯 (가스 버섯 + 스타푸르츠)
 - 겁쟁이 화분 (겁쟁이 버섯 + 화분)

- 오디세이/궁극 식물:
 - 하위종:
  - 버터 스쿼시위드 (스쿼시위드 + 버터위드)
  - 불꽃 황제 호두 (태양 황제 호두 + 할라페뇨)
  - 핵체리 스쿼시 (핵체리 폭탄 + 스쿼시)

 - 상위종:
  - 망각 멜론 대포 (망각 옥수수 대포 + 멜론 펄트)
  - 화산 양배추 대포 (화산 옥수수 대포 + 양배추 펄트)

 - 승급종:
  - 태양 대제 호두 (호두 + 태양 황제 호두)
  - 불꽃 대제 호두 (호두 + 불꽃 황제 호두 / 태양 대제 호두 + 할라페뇨)


- 유니크/미스테리 식물:
 - 양동이 (특정 식물들의 재료)
 - 미식축구 헬멧 (특정 식물들의 재료)
 - XXS 화분 (항아리깨기와 가챠 모드 한정 출현!!!)
 - 과욕 버섯 (전설의 궁극의 식물 카드)

2. 새로운 좀비들
- 삼지창 가르강튀아

2. 도감 업데이트
- 완전히 새롭게 개편된 도감을 이용하실 수 있습니다. 검색 기능과 업데이트된 구성 및 시각적 디자인이 포함되어 있습니다!
  (인게임 중 식물 위에 마우스 커서를 올린 상태에서 C 키를 누르면 해당 식물의 도감 정보를 즉시 확인할 수 있습니다.)

3. 새로운 가챠 모드 - 오디세이 가챠: 리워크
- 이 모드에서는 기본 식물들이 컨베이어 벨트에 나타납니다. 아래 숫자는 해당 식물을 심을 수 있는 횟수를 나타냅니다. 컨베이어 벨트에는 랜덤 버프가 있는 식물도 나타날 수 있습니다!

5. 나, 좀비 변경점
- 식판오분전 모드(나, 좀비 샌드박스)
- 이젠 모든 좀비들을 나, 좀비 샌드박스 모드에서 고를수 있도록 조정되었습니다.

4. 새로운 스킨
- 마법 고양이 둥지 > 할로윈날 고양이 둥지
- 중성자 스타푸르츠 > 마술사 스타푸르츠

7. 좀비 밸런스 조정
- 울트라 메카 호두
  - 체력: 90000 → 300000
  - 피해 제한 : 100 → 300
  - 대미지 흡수 방식: 이젠 모든 좀비(랜덤박스 좀비 제외)가 받는 피해가 1로 줄어드는 대신, 해당 피해 만큼 전부 자신이 피해받습니다.
- 특무장 가르강튀아
  - 버프:
- 외형은 더 이상 중요하지 않으며, 네 가지 능력을 동시에 지닙니다.

- 외형은 더 이상 중요하지 않으며, 네 가지 가르강튀아 모두 동일한 면역을 동시에 지닙니다.
